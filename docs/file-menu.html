File Menu

Main Menu

In the File Menu, you will find two important options.

Open Log Directory
This option will open your file browser to the WebTools-NG Log folder. The log folder will contain the log files produced by WebTools-NG during it's operation. These logs are particularly useful when reporting any issues you may be having with WebTools-NG. Please .zip the log file/s before submitting them to a WebTools-NG Github Issue Report, thanks.

Also see the Global Settings page to set the logging level as required.

Quit
This option should be obvious, it will Quit the program and close the WebTools-NG window.
