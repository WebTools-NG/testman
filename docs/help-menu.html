Help Menu

Main Menu

This menu contains a number of options for reaching out and getting help with WebTools-NG.

Forums
This option will take you to the WebTools-NG page on the Plex forums.

Github
This option will take you to the WebTools-NG Github page.

Help Wiki
This option will take you to the WebTools-NG Help Wiki page.

About WebTools-NG
This option will display the current WebTools-NG version number along with the platform type and it's architecture. The Copy button will also allow you to copy those details to the clipboard, so that you may paste the details into other applications, text files, posts etc...
